# muTOV
Microsecond Machine learning TOV emulator for Gravitational wave astrophysics 

## Installation instructions
```shell
git clone https://github.com/spencermagnall/muTOV.git
cd muTOV
pip install -r requirements.txt
pip install .
```

## Citation guide
